# 드라마 목록 JSON 형식 (dramas.json)

8천 편 이상 드라마를 넣을 때 사용하는 형식입니다.  
`assets/data/dramas.json` 파일을 이 형식으로 채우면 앱에서 자동으로 카드·상세 페이지에 반영됩니다.

## 파일 구조

```json
{
  "dramas": [
    { ... },
    { ... }
  ]
}
```

## 각 드라마 객체 필드

| 필드 | 필수 | 설명 | 예시 |
|------|------|------|------|
| **id** | ✅ | 고유 식별자 (문자열) | `"1"`, `"drama_001"` |
| **title** | ✅ | 제목 | `"청소부의 두 번째 결혼"` |
| **imageUrl** | | 사진 URL 또는 asset 경로. 비우면 플레이스홀더 | `"https://..."` 또는 `"assets/posters/1.jpg"` |
| **synopsis** | ✅ | 줄거리 | `"태성바이오 창립자 박창욱은..."` |
| **genre** | ✅ | 장르 (카드 부제·상세 장르에 사용) | `"비밀신분 · 로맨스"` |
| **episodes** | ✅ | 회차. **숫자** 또는 **배열** | `16` 또는 `[{"number":1,"title":"1화"}, ...]` |
| **cast** | ✅ | 배우 이름 배열 | `["김주연", "이민호"]` |

- **episodes**가 **숫자**면: 1화, 2화, … N화로 자동 생성 (제목: "1화", "2화", …).
- **episodes**가 **배열**이면: 각 항목에 `number`, `title`(선택) 사용. `title` 없으면 `"N화"`로 표시.

## 예시 (최소)

```json
{
  "dramas": [
    {
      "id": "1",
      "title": "드라마 제목",
      "synopsis": "줄거리 전문",
      "genre": "장르",
      "episodes": 20,
      "cast": ["배우1", "배우2"]
    }
  ]
}
```

## 엑셀에서 JSON 만들기

1. 엑셀 컬럼: `id`, `title`, `imageUrl`, `synopsis`, `genre`, `episodes`, `cast`  
   - **cast**는 한 셀에 `배우1, 배우2` 처럼 쉼표로 구분해 넣고, 로드 시 `split(',')`으로 배열로 변환하면 됨.
2. 엑셀 → CSV 저장 후, 온라인 변환기나 간단한 스크립트로 **JSON 배열**로 변환.
3. 변환된 배열을 `{ "dramas": [ ... ] }` 형태로 감싸서 `dramas.json`으로 저장.

파일 위치: **프로젝트 루트 기준** `assets/data/dramas.json`
